# Jobsity Video Test

Video Player based on a yarn monorepo with server and web packages.

## Development

Running:

```
yarn
yarn start
```

You'll get:
* Hot reloading enabled for [localhost](http://localhost:3000) environment
* Test graphql with [GraphiQL](http://localhost:8080/graphiql)
